Focal Points / Points Focaux Web Site (2019)

    Members:
  • Sherman Fong - coding
  • Laura Le - coding
  • Zahra Tabatabai - coding
    Materials:
  • HTML, CSS, Javascript

The Focal Points / Points Focaux web site was made to advertise Concordia University’s 2019 year-end D/CART show. The web site contains general information about the exhibition, the works were being exhibited in the show and the artists that are credited for the exhibited works. The challenge of the web site is to figure out how to make the exhibition's essential information retainable within a quick glance. Working with two other members of the web team, collaborations have been made with the exhibition’s visual identity team in order to ensure a consistent visual branding compared to other visual aspects of the show; such as the exhibition catalogue and the exhibition graphics itself.
