Assignment 2 - Portfolio Design Proposal - References

https://2mello.net/
I chose this example as one of my references for knowing how to display sound-based works in a portfolio. I also like the visual aesthetic of the web site; it is sleek, stylish and it pays homage to the visual identity of the Jet Set Radio video game series. The tribute of the game series also reflects in his works, in which both the original source materials and his portfolio works are based in a certain style in the hip-hop genre.

http://jukiokallio.com/index2.html
This is another example of someone's work portfolio that is primarily sound-based, which currently represents my curation of works for my portfolio. The inclusion of this example is meant to reference how the presentation of sound-based works are structured in a portfolio setting.

http://weirdcore.tv/av-portfolio/ (Epilepsy warning)
This example is included for aesthetic reasons and for structural reasons. I am inspired by the "weird" visual graphics of the website as well as how organized the works are structured by category (based on materials and genre), despite the visually chaotic nature.

https://deuxhuithuit.com/
This example is included for aesthetic reasons and for structural reasons. Similar to this web site, I want to be portrayed as someone who is easy-going and casual; while demonstrating my skills as a web developer at the same time.

https://www.kukuangyi.com/
This example is included mostly for how the works are documented in the web site within the digital arts context. The documentation of the works contained within this site are very in-depth, as evident by its description, credits and videos.

http://www.maotik.com/erratic/
This example is included mostly for how the works are documented in the web site within the digital arts context. Compared to the example of Kuang-Yi Ku, I chose this web site as a reference to see what a portfolio look like for a "smaller artist."
