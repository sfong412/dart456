TRACES: FLEUR-ESCENT (2017)

    MEMBERS:
  • CELESTE PIMM - concept
  • CHARLOTTE ROTHSCHING - physical development
  • DIMITRI KONTOGIANNOS - physical development
  • SHERMAN FONG - programming
    MATERIALS:
  • Carnations in vase of water, Raspberry Pi 3, water pump and fluorscein
  • Python 3

Traces: Fleur-escent is a collaborative art project created by Charlotte Rothsching, Celeste Pimm, Dimitri Kontogiannos and Sherman Fong. The interactive artwork is meant to represent the urgent issues of ecological contamination and data dissemination. Illuminated under a black light, Traces: Fleur-escent utilizes flowers that are being held in a glass vase; which is filled with water. Using the networking features of Raspberry Pi, the installation uses a Python script that is dependent on Twitter’s real-time API. The microcontroller to control the pump that outputs fluorscein (highlighter fluid) onto the water being held in the vase. The installation has been exhibited in You Are Here, Concordia’s D/CART year-end show of 2018.
